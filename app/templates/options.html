Chrome Extension

NoComment

Modernized controls for deciding where comment sections stay visible and how they disappear.

Current mode

Block only listed sites

NoComment will leave comments visible unless a page matches your Block List.

Default behaviour

Choose whether NoComment blocks comment sections everywhere by default, or only on sites you add to the Block List.

Hide style

Saved.

Pattern guide

Add hostnames or route patterns. Wildcards with * are supported and protocols are optional.

youtube.com/@mrbeast (all comments on all YouTube videos by MrBeast)

https://www.facebook.com/zuck/posts/* (all replies on all Facebook posts by Mark Zuckerberg)

https://x.com/brunomars/* (all replies on all X posts by Bruno Mars)

www.youtube.com/* (all comments on all YouTube videos)

x.com/*/status/* (all replies on all X posts)

www.reddit.com/r/news/* (all replies on all Reddit posts inside the r/news subreddit)

How it works

NoComment scans each page for common comment containers and hides them according to your saved rules.

Use the Allow List when you block comments everywhere by default, or the Block List when you only want to target selected sites.

The extension popup shows whether the current page is being changed right now.
